* 定义二维数组
  * 直接定义（不推荐）
    ```py
    a = [1, 2, 3]
    >>> print(a*3)               # 直接乘3，相当于把a1中元素乘3
    [1, 2, 3, 1, 2, 3, 1, 2, 3]
    >>> print([a]*3)             # 加上[], 将a1对象整体作为列表一个元素乘3
    [[1, 2, 3], [1, 2, 3], [1, 2, 3]]
    ```

        注意⚠：声明二维数组\[a\]\*3操作只是创建3个指向array的饮用，一旦修改arry改变，martix中3个list中元素也会随之改变

  * 间接定义（**推荐使用**）

  ```py
      matrix=[[ _ for i in range(3)] for _ in range(m)]
      >>> [[0, 0, 0], [1, 1, 1], [2, 2, 2]]
  ```
